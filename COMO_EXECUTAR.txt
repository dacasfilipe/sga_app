🚀 COMO EXECUTAR O SISTEMA SGA

═══════════════════════════════════════════

📦 ARQUIVO PRONTO: target\sga.war (27.6 MB)

═══════════════════════════════════════════

🔧 OPÇÕES PARA EXECUTAR:

1️⃣ COM XAMPP + TOMCAT:
   • Instalar XAMPP
   • Copiar sga.war para C:\xampp\tomcat\webapps\
   • Iniciar Tomcat no XAMPP
   • Acessar: http://localhost:8080/sga/

2️⃣ COM TOMCAT STANDALONE:
   • Baixar Apache Tomcat 9
   • Copiar sga.war para [TOMCAT]/webapps/
   • Executar [TOMCAT]/bin/startup.bat
   • Acessar: http://localhost:8080/sga/

3️⃣ COM SERVIDOR APPLICATION (GlassFish/WildFly):
   • Deploy do arquivo WAR
   • Configurar datasource PostgreSQL
   • Acessar na porta configurada
